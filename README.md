# Árbol Binario de Búsqueda (BST) – Fase 2

##  Descripción
Implementación en Java de un Árbol Binario de Búsqueda que permite inserción, búsqueda, eliminación y recorridos (In-Order, Pre-Order, Post-Order).

##  Archivos
- `BST.java`: Clase principal con la lógica del árbol.
- `Main.java`: Programa de prueba con inserciones y eliminaciones.

##  Métodos implementados
- `insert(int value)`
- `search(int value)`
- `delete(int value)`
- `inOrder(Node root)`
- `preOrder(Node root)`
- `postOrder(Node root)`
